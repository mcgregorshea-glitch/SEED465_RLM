# SEED Project Code

This repository contains the code for the SEED project.

## Structure

- `src/`: Contains the main source code.
  - `gcode_generator.py`: Generates G-code patterns.
  - `gcode_sender.py`: Sends G-code to the printer and monitors status.
- `legacy/`: Contains previous releases and development files.

## Usage

### G-Code Generator
Run the generator:
```bash
python src/gcode_generator.py
```

### G-Code Sender
Run the sender:
```bash
python src/gcode_sender.py
```
